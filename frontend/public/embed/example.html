üéØ How It Works

The CMP (Consent Management Platform) loads before any tracking scripts and blocks them until the user provides consent. Here's what happens:

  1. CMP SDK loads first - The embed script initializes
  2. Configuration fetched - Banner settings loaded from API
  3. Banner displayed - User sees consent options
  4. Scripts enabled - Only after user consent
  5. Consent recorded - Saved to backend database
